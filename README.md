# Wetter — Projekt

Dieses Repository enthält den Start für eine einfache, übersichtliche Wetter-Website (ein Ort). Der Fokus liegt auf schneller Verständlichkeit und einer schlanken UI.

Siehe auch: `IMPLEMENTATION_PLAN.md`.

Aufbau:

- `backend/` — Hinweise zur Einrichtung eines Laravel-Backends
- `frontend/` — kleine, statische Vue-Prototyp-Ansicht (CDN-basiert) für schnelles Testen

Nächste Schritte:

1. Backend (Laravel) initialisieren oder per API anbinden
2. Frontend erweitern und an Backend anpassen
